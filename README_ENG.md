# Werkzeug to Hashcat Converter (Mode 10900)

A fast and efficient Python script designed to convert hashes generated by the Werkzeug framework (commonly used in Flask applications) into a format fully compatible with Hashcat's mode 10900.

Ideal for Red Teaming engagements, CTFs, and security audits where you need to crack dumped passwords from Python-based web applications.

## 🚀 Features 

- Flexibility: Convert a single RAW hash directly from the CLI or bulk-process entire lists of hashes from a file.

- Automatic Encoding: Automatically parses and Base64-encodes the Salt and Hash, formatting them to Hashcat's strict structure (sha256:iters:b64_salt:b64_hash).

- Ready to Crack: Automatically generates and prints the exact Hashcat command you need, ready to copy and paste.

- Output Management: Option to save the converted hashes directly into a new text file.

## 🛠️ Usage

Convert a single hash
Use the -H flag to pass a single hash string.
```bash
python3 werkzeug2hashcat.py -H "pbkdf2:sha256:260000salthash_hex_here"
```
Process a list of hashes (Print to stdout)
Pass a text file containing one hash per line using the -f flag.
```bash
python3 werkzeug2hashcat.py -f dumped_hashes.txt
```
Process a list and save the results
Use -f for the input file and -o to define the output file for the formatted hashes.
```bash
python3 werkzeug2hashcat.py -f dumped_hashes.txt -o ready_to_crack.txt
```
